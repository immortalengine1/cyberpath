# CyberPath 🛡️
> Free offline cybersecurity career roadmap for beginners targeting their first security role.

## What is this?
A single HTML file that works as a complete study dashboard — no install, no account, no internet required after download.

## Features
- 📍 6-month week-by-week roadmap (Linux → Web Hacking → Security+ → First Job)
- ⏱ Study timer with session logging and 14-day progress chart
- 🃏 60 interview flashcards (Web Security, Networking, SOC, Pentesting, Security+)
- ⚡ XP + level system (Script Kiddie → Elite Hacker)
- 🔥 Daily streak tracker
- 💾 Everything saves in your browser automatically

## How to use
1. Download `index.html`
2. Open it in any browser
3. Start studying

## Built with
Pure HTML5, CSS3, and vanilla JavaScript. Zero dependencies.
